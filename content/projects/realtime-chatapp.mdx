A real‑time chat app built with Next.js 13 and fully typed using TypeScript.  
It uses Socket.IO Client for instant messaging and NextAuth for secure authentication.

The UI is styled with TailwindCSS and enhanced using Shadcn UI components. Together, these tools create a fast, modern, and developer‑friendly chat experience.
